TPE Updater





This website is a basic widget that can be used to streamline the player-updating experience. The idea behind this updater is two fold: (1) It takes some of the burden of messing up updates away from the player and places it on the website WHILE (2) giving updaters and auditors a uniform updating thread text. Both batters and pitchers updates are ready to go, with some minor bugs that can be worked out with roster-page changes by updaters/auditors.

What does this page do? It outputs a update post that looks like the one below. (This can be edited by request of the update/audit team)

What does this page NOT do? It does not link the URLs of a players activity to their update page, nor does it have any sort of validation checking. (This would still be a process saved for the update team)

Confirmed bug list

  • If player archetype is not set correctly, step 2 will break.
  • Step 2 cannot handle franctional updates.
  • If banked TPE is not set in one of a few ways, banked TPE will display as 0.
